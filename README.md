# AppImages Downloader

## A simple application for downloading appimages.

---

[Download](https://github.com/hawier-dev/appimages-dl/releases/latest)

---

### Usage

##### Update repo:

```
appimgdl updaterepo
```

##### Download Appimage:

```
appimgdl get <appimage>
```

##### Remove Appimage:

```
appimgdl remove <appimage>
```
