# AppImages Downloader

A simple application for downloading appimages.

### Installing

```
curl https://raw.githubusercontent.com/hawier-dev/appimages-dl/main/install.sh | bash
```

### Usage

##### Update repo:

```
appimgdl updaterepo
```

##### Download Appimage:

```
appimgdl get <appimage>
```

##### Remove Appimage:

```
appimgdl remove <appimage>
```
